🧬 Why this curriculum

This site was built to accompany R&D work on infectious-pathogen in-vitro diagnostics (IVDs) — the kind of project that combines microbiology, molecular biology, and modern data science. The 21 days are sequenced so that by the end you can read primer-design literature, run DESeq2 on a host-response dataset, build a Snakemake pipeline for pathogen detection, and ship the whole thing to GitHub.

Stack covered: R (Bioconductor, DESeq2, Seurat, Biostrings, GenomicRanges, ggplot2), Python (Biopython, pandas, scikit-learn, ESM, Snakemake), shell (samtools, bcftools, bedtools), and the canonical public databases (NCBI, Ensembl, UniProt, PDB).
